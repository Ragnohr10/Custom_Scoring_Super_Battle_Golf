# Custom Scoring

Custom scoring system for **Super Battle Golf**.

## Features
- Custom placement points (1st to 8th place)
- Custom stroke points (1 to 10+ strokes)
- Enable/disable via in-game menu
- Compatible with French and English

## Installation
Place this package into the game folder so the DLL ends up at `BepInEx/plugins/Custom_Scoring/Custom_Scoring.dll`.

## Version
- `0.1.0`
